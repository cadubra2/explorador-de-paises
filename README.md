# Explorador de Países — REST Countries API

## Funcionalidades
- Buscar países por nome
- Filtrar por continente
- Exibir:
  - Bandeira
  - Capital
  - População
  - Área
  - Idiomas
  - Moedas
  - Mapa
- Sistema de favoritos com localStorage (página dedicada)
- Página de detalhes
- Página Sobre
- Tema claro/escuro persistente (dark mode)
- Design moderno e responsivo (melhorias mobile)
- Tratamento de erros + spinner

## Arquivos principais
- `index.html` — página principal (busca, filtro)
- `favorites.html` — lista de favoritos (remover e abrir detalhes)
- `detalhes.html` — detalhes do país (mapa embed)
- `sobre.html` — sobre o projeto e autores
- `css/style.css` — estilos (inclui dark mode)
- `js/app.js` — lógica principal (busca, filtro, favoritos iniciais)
- `js/detalhes.js` — lógica de detalhes do país
- `js/favorites.js` — lógica da página de favoritos

## API
REST Countries — https://restcountries.com/
